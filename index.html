Clinical Intelligence Platform (CLIP)

Problem

Clinical workflows relied on fragmented datasets, making decision-making slow, error-prone, and cognitively heavy.

  • Disconnected data sources
  • High cognitive load
  • Dependency on technical users

Goal

Enable faster, insight-driven clinical decisions using AI.

Approach

Mapped workflows with clinicians and identified breakdowns in data interpretation. Focused on simplifying complexity.

Solution

Designed an AI-powered platform transforming raw clinical data into actionable insights.

Impact

  • Reduced decision-making time
  • Improved usability across teams
  • Enabled scalable AI workflows

Key Decisions

  • Insight-first design
  • Progressive disclosure
  • AI as assistive layer

Learnings

  • Simplification drives adoption
  • Trust is critical in AI UX

Search Experience (Expedia Group)

Problem

Search is the highest-impact entry point but suffered from complexity and drop-offs.

  • High friction in input fields
  • Poor clarity in guest vs room selection
  • Low conversion rates

Goal

Improve search usability and increase booking conversion.

Approach

Conducted usability testing and analyzed behavioral data to identify friction patterns.

Solution

Redesigned search form with simplified inputs, better defaults, and clearer hierarchy.

Impact

  • Improved form completion rate
  • Reduced user confusion
  • Enhanced cross-platform consistency

Key Decisions

  • Room-first vs guest-first logic
  • Smart defaults
  • Reduced cognitive load

Learnings

  • Small UX changes can drive big revenue impact
  • Clarity beats flexibility

Simulation Input Manager

Problem

Enterprise users struggled with complex simulation configurations and inputs.

  • Overwhelming forms
  • Lack of guidance
  • High error rates

Goal

Simplify complex enterprise workflows.

Approach

Studied expert workflows and identified patterns for structuring inputs logically.

Solution

Designed modular input system with guided flows and validation.

Impact

  • Reduced user errors
  • Faster task completion
  • Improved adoption

Key Decisions

  • Step-by-step flows
  • Context-aware inputs
  • Error prevention

Learnings

  • Enterprise UX needs structure over flexibility
  • Guidance reduces cognitive load
